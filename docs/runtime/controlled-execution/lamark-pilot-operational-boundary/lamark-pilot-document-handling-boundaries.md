# Lamark Pilot Document Handling Boundaries

## Purpose

This document defines document handling boundaries for the Lamark controlled pilot.

## Allowed Document Handling

- Define classification rules.
- Classify approved sandbox copies.
- Record document evidence references.
- Map document categories for knowledge management.
- Review approved metadata.
- Correlate document evidence with audit requirements.

## Actions Requiring Explicit Authorization

- Reading real Lamark documents.
- Classifying real Lamark documents.
- Moving real documents.
- Renaming real documents.
- Deleting real documents.
- Sharing real documents.
- Sending real document contents.
- Transforming real documents.
- Modifying document permissions.

## Blocked Without Authorization

Real documents must not be moved, deleted, shared, sent, permission-modified, or transformed without explicit human authorization.

## Handling Rule

Sandbox classification does not authorize handling of real documents. Evidence records document handling context; they do not grant permission.
