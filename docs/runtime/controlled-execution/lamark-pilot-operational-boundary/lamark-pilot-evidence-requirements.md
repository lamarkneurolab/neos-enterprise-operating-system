# Lamark Pilot Evidence Requirements

## Purpose

This document defines evidence required for the Lamark controlled pilot.

## Required Evidence

| Evidence Type | Required For |
|---|---|
| Boundary evidence | Pilot scope, allowed actions, and blocked actions |
| Authorization evidence | Any action requiring human authorization |
| Data access evidence | Any approved data or metadata access |
| Document handling evidence | Any approved real document review or classification |
| Sensitive information review evidence | Any suspected sensitive information |
| Sandbox evidence | Approved sandbox activity |
| Audit evidence | Audit event creation and correlation |
| Rollback evidence | Rollback trigger, scope, and outcome |
| Incident evidence | Incident detection, severity, block status, and closure |
| Handoff evidence | Source owner, target owner, open items, and acceptance |

## Evidence Rules

- Evidence records do not authorize execution.
- Evidence correlates pilot activity with review requirements.
- Missing evidence blocks readiness review.
- Historical evidence is not current authorization.
- Evidence must be linked to audit requirements when applicable.
