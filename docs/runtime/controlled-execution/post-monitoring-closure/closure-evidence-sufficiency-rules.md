# Closure Evidence Sufficiency Rules

## Purpose

Define when evidence is sufficient for documentary closure.

## Minimum evidence

- Authorization evidence.
- Scope evidence.
- PR, branch, head SHA, base SHA and commit evidence.
- Runtime window evidence.
- Monitoring boundary evidence.
- Closure boundary evidence.
- Result validation evidence.
- Audit closure evidence.
- Incident evidence when applicable.
- Rollback evidence when applicable.
- Unresolved risk evidence.
- Human closure approval evidence.

## Blocking missing evidence

Closure is blocked when authorization, scope, SHA, PR, branch, audit, incident,
rollback, unresolved risk or human approval evidence is missing or inconsistent.

## Optional evidence

Additional notes, screenshots, summaries or lessons learned may support review
but cannot replace required evidence.

## Sufficiency rule

Evidence is sufficient only when it is current, scoped, traceable, auditable and
able to justify the closure decision.
