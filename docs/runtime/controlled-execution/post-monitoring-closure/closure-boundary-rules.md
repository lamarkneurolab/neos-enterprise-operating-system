# Closure Boundary Rules

## Purpose

Define what documentary closure allows and prohibits.

## Closure allows

- Recording final post-monitoring review.
- Recording final evidence sufficiency.
- Recording audit closure.
- Recording rollback outcome.
- Recording incident linkage.
- Recording unresolved risk decisions.
- Recording human closure approval.
- Handing off documentary context to a future block.

## Closure prohibits

- Production activation.
- Runtime execution.
- Business, commercial, clinical, documentation, marketing, sales or operations
  agents.
- External integrations.
- Runners or CI.
- Databases or external services.
- Permission changes.
- Secret management.
- Dependency installation.
- File deletion.
- Destructive changes.
- Automatic merge.

## Boundary rule

Closure is documentary only and cannot expand scope beyond the approved closure
boundary.
