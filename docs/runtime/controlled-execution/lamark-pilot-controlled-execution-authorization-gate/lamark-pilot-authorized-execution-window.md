# Lamark Pilot Authorized Execution Window

## Purpose

This document defines the authorized execution window requirement for a future Lamark controlled pilot.

## Window Requirements

An authorized execution window must include:

- Start date and time.
- End date and time.
- Approver.
- Authorized scope.
- Authorized operator or accountable owner.
- Evidence package reference.
- Rollback readiness reference.
- Incident escalation owner.
- Expiration condition.

## Window Boundary

Defining an execution window does not activate the window. Execution remains blocked until a separate future action explicitly authorizes execution within the documented window.

## Invalid Window Conditions

The window is invalid if it is open-ended, lacks human approval, lacks rollback readiness, lacks incident escalation ownership, or includes production activation outside the controlled pilot boundary.
