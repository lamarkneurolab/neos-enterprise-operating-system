# Lamark Pilot Document Handling Preconditions

## Purpose

This document defines document handling preconditions for the Lamark pilot authorization gate.

## Required Preconditions

Authorization requires confirmation that:

- Only authorized documentation is used.
- Any sandbox copy is authorized and traceable.
- Source document identifiers are recorded when applicable.
- Redaction status is recorded when applicable.
- Sensitive-data review status is recorded when applicable.
- Document handling scope is explicit.

## Prohibited Handling

Without separate human authorization, the pilot must not move, delete, share, send, permission-modify, or transform real documents.

## Failure Rule

If document handling is unauthorized, untraceable, or dependent on real document changes, authorization is denied.
