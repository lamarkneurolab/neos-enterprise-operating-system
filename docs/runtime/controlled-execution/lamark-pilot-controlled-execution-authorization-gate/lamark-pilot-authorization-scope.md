# Lamark Pilot Authorization Scope

## Purpose

This document defines authorized and prohibited scope for the Lamark pilot authorization gate.

## Authorized Scope

The gate may document:

- Authorization decision scope.
- Go/no-go decision criteria.
- Human approval record requirements.
- Authorized execution window requirements.
- Execution boundary confirmation.
- Evidence preconditions.
- Sandbox readiness preconditions.
- Data boundary preconditions.
- Document handling preconditions.
- Incident blocker preconditions.
- Rollback preconditions.
- Audit correlation preconditions.
- Handoff preconditions.
- Authorization denial conditions.
- Final authorization gate checklist.

## Prohibited Scope

The gate must not authorize production activation, real customer use, external sales, external commercial diagnosis, sensitive financial data use, autonomous action, permission changes, real document movement, real document deletion, real document sharing, information sending, or real document transformation.

## Scope Rule

Anything not explicitly included remains prohibited and blocked until a future separately authorized action exists.
