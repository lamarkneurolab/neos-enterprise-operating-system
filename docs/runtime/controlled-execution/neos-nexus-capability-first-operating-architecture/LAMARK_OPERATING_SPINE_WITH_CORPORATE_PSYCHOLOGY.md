# Lamark Operating Spine with Corporate Psychology

## Purpose

This document defines the minimum Lamark operating spine for NEOS/NEXUS with Corporate Psychology as a core operational domain from day one.

## Minimum Domains

- Sales and commercialization.
- Decision architecture.
- Corporate Psychology.
- NeuroAudit(TM).
- Content production.
- PMO / operations.
- QA, evidence, and governance.

6. Internal request.

## Minimum Corporate Psychology Flow

Organizational request
-> Need classification
-> Ethical risk / confidentiality
-> Organizational brief
-> Instrument or methodology definition
-> Consent / boundaries
-> Information collection or analysis
-> Aggregated findings
-> Organizational recommendations
-> Tiziano review
-> Final deliverable
-> Learning to Knowledge Base

## Sensitivity Classification

Corporate Psychology is a sensitive C5 domain when it uses real psychological or organizational data.

Any use of real psychological data, organizational diagnostics, individual-level records, sensitive behavioral evidence, or confidential human data requires explicit authorization and must follow ethics, confidentiality, aggregation, and evidence controls.

## Lamark Internal Production

Lamark Internal Production is not a pilot, demo, or decorative sandbox. It is the first real internal operating environment of NEOS/NEXUS.

Internal production still requires switches, authorization boundaries, evidence, and escalation for critical impact.
