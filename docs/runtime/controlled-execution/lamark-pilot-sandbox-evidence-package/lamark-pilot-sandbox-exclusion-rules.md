# Lamark Pilot Sandbox Exclusion Rules

## Purpose

This document defines what must remain excluded from the Lamark documentary sandbox.

## Excluded Activities

The sandbox excludes:

- Production activation.
- Runtime execution.
- External sales.
- Real customer workflows.
- External commercial diagnosis.
- Autonomous actions.
- New agents.
- Real external integrations.
- Runners.
- New CI.
- Databases.
- External services.
- Permission administration.
- Secret administration.
- Dependency installation.
- Destructive changes.
- File deletion.
- Pull request creation.
- Merge activity.

## Excluded Document Actions

The sandbox excludes moving, deleting, sharing, sending, transforming, or changing permissions on real documents unless a future separate human authorization explicitly permits that action.

## Excluded Data

Sensitive financial data, customer data, unredacted personal data, credentials, secrets, payment data, banking data, health diagnosis data, and legally restricted data remain blocked.
