# Lamark Pilot Sandbox Document Copy Rules

## Purpose

This document defines rules for authorized sandbox document copies.

## Copy Rule

Only authorized sandbox copies may be used for Lamark pilot preparation. A sandbox copy is valid only when a human approver has authorized the copy purpose, source, scope, retention expectation, and permitted handling.

## Required Copy Metadata

Each authorized sandbox copy must record:

- Source document identifier.
- Copy creation date.
- Human approver.
- Authorization reference.
- Allowed use.
- Redaction status.
- Sensitive-data review status.
- Retention or disposal expectation.

## Prohibited Copy Actions

Without separate human authorization, sandbox preparation must not:

- Move real documents.
- Delete real documents.
- Share real documents.
- Send real documents or document contents.
- Modify document permissions.
- Transform real documents.
- Replace real documents with sandbox copies.
