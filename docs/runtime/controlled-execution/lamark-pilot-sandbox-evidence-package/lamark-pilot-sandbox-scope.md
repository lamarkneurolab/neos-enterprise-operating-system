# Lamark Pilot Sandbox Scope

## Purpose

This document defines the included scope for the Lamark documentary sandbox.

## Included Scope

The sandbox includes only preparation activities required to evaluate controlled pilot readiness:

- Documentary sandbox setup requirements.
- Authorized sandbox-copy rules.
- Allowed data categories.
- Sensitive data blocklist.
- Human authorization checkpoints.
- Minimum evidence package requirements.
- Minimum evidence records.
- Audit-correlation requirements.
- Readiness controls.
- Rollback triggers.
- Incident blockers.
- Handoff requirements.
- Acceptance checklist.

## Controlled Pilot Boundary

Lamark remains a controlled pilot. Inclusion in this scope does not convert the pilot into production, does not permit open customer usage, and does not permit external commercial diagnosis.

## Excluded By Default

Anything not explicitly included in this documentary scope remains excluded and blocked until separately authorized.
