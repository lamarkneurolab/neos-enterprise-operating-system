# Lamark Pilot Sandbox Minimum Evidence Records

## Purpose

This document defines the minimum evidence records required for Lamark sandbox preparation.

## Minimum Records

| Record | Required Content |
|---|---|
| Sandbox scope record | Included scope, exclusions, version, owner |
| Copy authorization record | Source, copy purpose, approver, date, allowed use |
| Data allowance record | Data category, sensitivity review, approval status |
| Blocklist verification record | Confirmation that blocked data is absent |
| Human authorization record | Approver, action, scope, evidence reference |
| Audit-correlation record | Evidence ID, audit event ID, decision linkage |
| Readiness control record | Control, status, reviewer |
| Rollback trigger record | Trigger, condition, response owner |
| Incident blocker record | Blocker, severity, stop condition |
| Handoff record | Recipient, open items, evidence package link |
| Acceptance record | Checklist status, reviewer, date |

## Completeness Rule

If any minimum record is missing, incomplete, or unreviewed, sandbox readiness remains blocked.
