# Lamark Pilot Sandbox Inclusion Rules

## Purpose

This document defines what may be included in the Lamark documentary sandbox.

## Inclusion Rules

An item may be included only when all of the following are true:

- It is needed for controlled pilot preparation.
- It is documentary, evidentiary, or governance-oriented.
- It does not activate production or runtime behavior.
- It does not involve real customers.
- It does not contain sensitive financial data.
- It does not require new external integrations, agents, runners, CI, databases, or services.
- It uses an authorized sandbox copy when document content is needed.
- It has human authorization when authorization is required by this package.

## Allowed Sandbox Materials

- Approved taxonomy drafts.
- Approved classification examples.
- Redacted or synthetic sample records.
- Authorized sandbox document copies.
- Evidence register templates.
- Audit-correlation templates.
- Readiness checklist records.
- Handoff notes that contain no blocked sensitive data.

## Inclusion Control

If an item cannot be clearly classified as allowed, it remains excluded until a human approver authorizes inclusion in writing.
