# Lamark Pilot Evidence Source Traceability

## Purpose

This document defines source traceability requirements for Lamark pilot evidence.

## Source Traceability Requirements

Each evidence record must trace to a known source with:

- Source identifier.
- Source owner.
- Source type.
- Source date.
- Authorization reference for source use.
- Sandbox copy reference when applicable.
- Redaction or sensitivity review status.
- Reviewer who confirmed traceability.

## Allowed Source Types

Allowed source types include:

- Authorized sandbox copies.
- Synthetic examples.
- Redacted documentary examples.
- Approved metadata records.
- Approved audit records.
- Approved authorization records.
- Approved handoff records.

## Blocked Source Types

Sources are blocked when they contain real customer data, sensitive financial data, secrets, credentials, unapproved personal data, production-only data, or material requiring permission changes.
