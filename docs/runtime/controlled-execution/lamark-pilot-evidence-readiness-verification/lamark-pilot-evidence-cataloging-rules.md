# Lamark Pilot Evidence Cataloging Rules

## Purpose

This document defines cataloging rules for Lamark pilot evidence.

## Cataloging Rules

Evidence must be cataloged by:

- Evidence ID.
- Evidence category.
- Source type.
- Sandbox scope area.
- Readiness control area.
- Authorization dependency.
- Audit correlation dependency.
- Data sensitivity status.
- Document copy status.
- Review status.

## Evidence Categories

Allowed evidence categories are:

- Sandbox scope evidence.
- Human authorization evidence.
- Data boundary evidence.
- Document copy evidence.
- Audit correlation evidence.
- Completeness evidence.
- Incident blocker evidence.
- Rollback readiness evidence.
- Handoff evidence.
- No-go condition evidence.

## Cataloging Control

Evidence that cannot be cataloged clearly must remain blocked until a human reviewer assigns the correct category and confirms that it is allowed for documentary use.
