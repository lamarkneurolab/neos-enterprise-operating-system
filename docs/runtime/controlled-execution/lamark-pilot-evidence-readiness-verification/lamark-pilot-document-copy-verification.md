# Lamark Pilot Document Copy Verification

## Purpose

This document defines verification of sandbox document copies for Lamark pilot evidence.

## Required Copy Verification

Each document-copy evidence record must confirm:

- The copy is an authorized sandbox copy.
- The source document identifier is recorded.
- The copy purpose is recorded.
- The approver is recorded.
- The authorization reference is recorded.
- The allowed use is recorded.
- Redaction status is recorded.
- Sensitive-data review status is recorded.

## Prohibited Real Document Actions

Verification must confirm that no real documents were moved, deleted, shared, sent, transformed, or permission-modified without explicit human authorization.

## Failure Rule

If a document copy is unauthorized, untraceable, unreviewed, or derived from a real document action without authorization, readiness verification fails.
