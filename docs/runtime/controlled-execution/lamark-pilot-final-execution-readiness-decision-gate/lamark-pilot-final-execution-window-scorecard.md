# Lamark Pilot Final Execution Window Scorecard

## Purpose

This scorecard defines final execution window readiness criteria for the Lamark pilot readiness decision gate.

## Scorecard

| Criterion | Required Condition | Gate Impact |
|---|---|---|
| Window definition | Any future window is documented as proposed only | Required |
| Start condition | No start condition is activated by readiness | Blocking |
| End condition | Proposed end condition is documented for future review | Required |
| Human presence | Required human presence is documented for any future execution request | Required |
| Stop authority | Stop authority and escalation owner are documented | Required |
| Window conflict | Conflicts or unavailable coverage produce defer or no-go | Blocking |

## Passing Rule

Execution window readiness passes only when a future proposed window is bounded, staffed, reversible, and not activated by this gate.

## Failure Rule

Any active start signal, unbounded window, missing human coverage, missing stop authority, or production activation fails this scorecard.
