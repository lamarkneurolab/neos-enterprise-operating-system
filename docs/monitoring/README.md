# Monitoring & Observability

Health checks, logs, telemetry, alerting and audit observability.

## Controlled execution monitoring

NEOS Fase 3 Block 14 defines documentary controlled execution monitoring and
live safeguards in `../runtime/controlled-monitoring/README.md`.

The Block 14 documents define monitoring boundaries, health observation, safety
signals, live evidence, audit linkage, drift detection, anomaly handling, stop
conditions, emergency pause, human escalation, rollback triggers and monitoring
handoff rules. They do not execute runtime, monitor production or activate
services.
