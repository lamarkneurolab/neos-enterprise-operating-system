# Changelog

All notable changes to NEOS will be documented here.

## [0.1.0] - 2026-06-25

### Added

- Initial repository structure package for NEOS Fase 3.
- Governance files.
- ADR directory.
- Evidence and audit logs.
- Registry placeholders.
- DevOps and execution layer documentation structure.
- Registry System v0.1.0 contract.
- Canonical registry records for tools, connectors, capabilities, permissions and runtimes.

### Governance

- GitHub established as canonical repository.
- Markdown established as official source format.
- Google Drive established as visual/documentary mirror only.
- Registry records require ownership, authorization and evidence metadata.
